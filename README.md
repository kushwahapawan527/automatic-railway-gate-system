## ⚙️ Features
- Real-time train detection using IR sensors  
- Automatic gate control using servo motor  
- Reduces human error and improves safety  

## 🛠️ Components
- Arduino Uno  
- IR Sensor  
- Servo Motor  
- Power Supply
